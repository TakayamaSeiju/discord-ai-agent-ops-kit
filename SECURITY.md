# Security Policy

Thank you for helping keep Discord AI Agent Ops Kit safe and trustworthy.

This project provides templates and workflows for using AI coding agents in open-source and small-team development. Because some workflows may involve code, documentation, prompts, or operational processes, we care about security, privacy, and responsible AI use.

## Reporting a security issue

If you find a security issue, please do not open a public issue with sensitive details.

Instead, please contact the maintainer directly.

Maintainer: Seiju Takayama  
Email: takayama@totonoete.jp

Please include:

- A clear description of the issue
- Steps to reproduce, if applicable
- Potential impact
- Suggested mitigation, if you have one

## Scope

Security-related topics may include:

- Unsafe AI agent workflows
- Prompts that may expose sensitive information
- Documentation that encourages insecure behavior
- Templates that could cause accidental leakage of secrets
- Examples that include private tokens, credentials, or personal data
- Recommendations that could create privacy or security risks

## Responsible AI use

Contributors should avoid submitting examples that include:

- API keys
- Passwords
- Private tokens
- Personal information
- Customer data
- Confidential business information
- Security-sensitive internal workflows

AI-generated content should be reviewed by a human before being committed.

## Supported versions

This project is in early development. Security updates will focus on the latest version of the repository.

## Maintainer response

The maintainer will review security reports and update documentation, templates, or examples when needed.
