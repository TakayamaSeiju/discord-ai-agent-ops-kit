# Codex Task Request Template

Use this template when asking Codex or another AI coding agent to work on a development or documentation task.

## Task summary

Describe the task in one or two sentences.

## Background

Explain why this task matters and what problem it solves.

## Files to inspect

List the files or folders that the AI agent should review first.

- README.md
- docs/
- templates/

## Expected output

Describe what the AI agent should produce.

Examples:

- Update documentation
- Draft a pull request
- Review an issue
- Generate release notes
- Improve a workflow template

## Constraints

Add any rules, limitations, or things the agent should avoid.

## Review checklist

Before accepting the output, check that:

- The result is easy to understand
- The change is small enough to review
- Documentation is updated if needed
- No unnecessary complexity is added
- The maintainer can verify the result
