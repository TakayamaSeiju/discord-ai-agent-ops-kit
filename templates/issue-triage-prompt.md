# Issue Triage Prompt

Use this prompt to ask Codex or another AI assistant to help classify and summarize open-source issues.

## Prompt

You are helping maintain an open-source repository.

Please review the issue and classify it as one of the following:

- bug
- documentation
- feature request
- question
- maintenance
- security

Then provide:

1. A short summary of the issue
2. Missing information, if any
3. Suggested labels
4. Suggested next action
5. Whether Codex can help prepare a fix
6. Any possible security or privacy concerns

## Review checklist

Before acting on the AI output, the maintainer should confirm:

- The issue was classified correctly
- The suggested labels are appropriate
- The next action is realistic
- No sensitive information is exposed
- The response is helpful to contributors
