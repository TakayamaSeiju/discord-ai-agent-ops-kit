# Code of Conduct

## Our commitment

Discord AI Agent Ops Kit is an open-source project for people who want to use AI coding agents more effectively and responsibly.

We are committed to providing a welcoming, respectful, and productive environment for all contributors, including solo developers, small teams, educators, community builders, non-engineers, and open-source maintainers.

## Expected behavior

Participants are expected to:

- Be respectful and constructive
- Welcome different experience levels
- Give clear and helpful feedback
- Focus on improving the project
- Respect privacy and confidentiality
- Review AI-generated content carefully before sharing it

## Unacceptable behavior

Unacceptable behavior includes:

- Harassment, intimidation, or personal attacks
- Discriminatory language or behavior
- Publishing private information without permission
- Sharing API keys, passwords, tokens, or confidential data
- Submitting unsafe or misleading AI-generated content
- Repeatedly disrupting discussions or contribution processes

## Responsible AI use

This project encourages responsible use of AI coding agents.

Contributors should make sure that AI-generated content is:

- Reviewed by a human
- Accurate enough for practical use
- Free from sensitive information
- Helpful to maintainers and contributors
- Aligned with the project’s goals

## Enforcement

The maintainer may remove content, close issues, reject pull requests, or limit participation if behavior harms the project or community.
