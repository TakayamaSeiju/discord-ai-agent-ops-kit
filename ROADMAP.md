# Roadmap

This roadmap outlines the planned development of Discord AI Agent Ops Kit.

The goal of this project is to provide practical, reusable workflows for small open-source teams, solo developers, educators, and community builders who want to use AI coding agents such as Codex more effectively.

## v0.1 - Initial maintainer workflow templates

* Add basic Codex task request template
* Add issue triage prompt
* Add pull request review checklist
* Add README with project overview
* Add MIT License
* Define the initial project scope

## v0.2 - Discord-based agent operation examples

* Add Discord channel setup example
* Add Discord-based task delegation workflow
* Add examples for solo developers
* Add examples for small OSS teams
* Add maintainer weekly review template

## v0.3 - Documentation and release workflows

* Add documentation update workflow
* Add release note generation template
* Add changelog preparation checklist
* Add contributor onboarding guide
* Improve examples for non-engineers coordinating software projects

## v0.4 - Responsible AI agent use

* Add responsible AI agent use guide
* Add security and privacy checklist
* Add review guidelines for AI-generated code
* Add workflow examples for educators and community builders

## Long-term direction

The long-term goal is to make AI-assisted open-source maintenance more practical, transparent, and repeatable.

This project will focus on:

* issue triage
* pull request review preparation
* documentation quality
* release workflows
* maintainer automation
* Discord-based coordination
* responsible use of AI coding agents
